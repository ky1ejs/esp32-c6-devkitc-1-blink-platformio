# ESP32-C6 Arduino PlatformIO
This is a working example of the ESP32-C6 with Arduino using PlatformIO. It shows an example using the Adafruit OLED FeatherWing and some [debug scanning for I2C devices that I found on reddit](https://www.reddit.com/r/esp32/comments/1ca6y27/comment/l0u8x22/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button).
